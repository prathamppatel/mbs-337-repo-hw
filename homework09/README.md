# Homework 09 - Linear Classification of Breast Cancer Malignancy

A Jupyter notebook that builds a linear classifier to predict whether a breast cancer tumor is benign or malignant using the UCI Breast Cancer Wisconsin Dataset and scikit-learn.

## Setup Instructions

Run these commands to launch the Jupyter server:
```bash
cd $HOME/mbs-337
source myenv/bin/activate
jupyter lab
```

Then open a browser and navigate to:
http://<your-vm-public-ip>:8888/lab

Open `Notebook.ipynb` from the file browser on the left and run the cells.

## Notebook Overview

- **Part 1** - Load and examine the breast cancer dataset
- **Part 2** - Prepare and split the data into training and test sets
- **Part 3** - Fit a linear classifier using the Perceptron algorithm
- **Part 4** - Validate the model and assess performance with accuracy scores and a confusion matrix
